1️⃣ トランプ大統領だいとうりょう「イランの発電所はつでんしょへの攻撃こうげきは5日いつか間かんしない」

イランに近ちかいホルムズ海峡かいきょうでは、今いまも石油せきゆを運はこぶたくさんの船ふねが通とおることができません。
トランプ大統領だいとうりょうは、今月こんげつ21日にちSNSに、船ふねがホルムズ海峡かいきょうを通とおることができるようにしなければ48時間じかん以内いないにイランの発電所はつでんしょなどを攻撃こうげきすると書かきました。
しかし、23日にちには「イランといい話はなし合あいができた」とSNSに書かきました。そして、攻撃こうげきを5日いつか間かんしないことにしたと言いいました。
これについてイランは「話はなし合あいはしていない。アメリカは嘘うそをついている。石油せきゆの値段ねだんが上あがって、戦たたかいも長ながくなって、困こまっているからだ」と言いっています。

2️⃣ スーパーの米こめ　平均へいきんの値段ねだんが5kgで4000円えんより安やすくなった

最近さいきん、米こめの値段ねだんが安やすくなってきました。
国くには、毎週まいしゅうスーパーで売うっている米こめの値段ねだんを調しらべています。
今月こんげつ15日にちまでの1週間しゅうかんの平均へいきんの値段ねだんは、5kgで3980円えんでした。前まえの1週間しゅうかんより33円えん安やすくなりました。
5週間しゅうかん続つづけて下さがりました。前まえに米こめの値段ねだんが4000円えんより安やすかったのは、去年きょねんの8月がつでした。
国くには「スーパーに米こめを売うっている会社かいしゃが、持もっている米こめを早はやく売うるために値段ねだんを下さげているからではないか」と考かんがえています。

3️⃣ 4月がつから6月がつは気温きおんが高たかくなりそう　6月がつの大雨おおあめに気きをつけて

気象庁きしょうちょうは、4月がつから6月がつまでの天気てんきがどうなるか発表はっぴょうしました。
日本にっぽんはどの地方ちほうも、気温きおんが高たかくなりそうです。5月がつには30℃以上いじょうになる日ひがありそうです。
雨あめは、関東地方かんとうちほうから九州きゅうしゅうまでは、いつもの年としと同おなじか多おおく降ふりそうです。沖縄おきなわや奄美あまみでは、いつもの年としと同おなじか少すくなくなりそうです。この冬ふゆ雨あめが少すくなかった所ところでは、山火事やまかじなどに気きをつけてください。
梅雨つゆの6月がつごろは、いつもの年としより雨あめが多おおいかもしれません。
気象庁きしょうちょうは、暑あつさやたくさんの雨あめへの準備じゅんびを早はやく始はじめるように言いっています。

4️⃣ 大阪おおさかのまちの中なかに鹿しかが出でた

大阪おおさかのまちの中なかに鹿しかが出でました。
22日にちの朝あさ、大阪おおさかで「鹿しかが歩あるいている」という連絡れんらくが警察けいさつにありました。
鹿しかがいたのはJＲ大阪駅おおさかえきから3kmぐらいの家いえがたくさんある所ところです。
23日にちと24日にじゅうよっかも、大阪市おおさかし内ないのほかの場所ばしょで鹿しかを見みた人ひとがいました。同おなじ鹿しかかどうかは分わかっていません。
困こまったことは起おこっていないので、市役所しやくしょでは捕つかまえてはいません。
隣となりの奈良県ならけんには、鹿しかがたくさんいる奈良公園ならこうえんがあるので、そこから来きたかもしれないと言いう人ひともいます。
専門家せんもんかは「小ちいさい角つのがあって危あぶないので近ちかづかないでください」と話はなしています。
