1️⃣ イランの戦たたかいは止とまったがレバノンでは攻撃こうげきが続つづく

日本にっぽんの時間じかんの8日ようか、アメリカとイランは、2週間しゅうかん、戦たたかいを止とめることにしました。
11日にちに、パキスタンで、戦たたかいを全部ぜんぶ終おわりにするための話はなし合あいをする予定よていです。
アメリカは、バンス副ふく大統領だいとうりょうが出でます。
バンス副ふく大統領だいとうりょうは「イランが核兵器かくへいきを作つくらないようにすることが、話はなし合あいの大切たいせつな目的もくてきの一ひとつだ」と言いいました。
しかし、レバノンでは、「ヒズボラ」とイスラエルが戦たたかっています。ヒズボラはイスラム教きょうのグループで、イランが活動かつどうを助たすけています。
8日ようかのイスラエルの攻撃こうげきで、たくさんの人ひとが亡なくなったと、メディアは伝つたえています。
イランは、レバノンの戦たたかいも止とめなければいけない、と言いっています。

2️⃣ 北海道ほっかいどう 熊くまから人ひとを守まもるため電気でんきが流ながれる柵さくを置おいた

冬ふゆのあいだ眠ねむっていた熊くまは、春はるになると、目めが覚さめて動うごき始はじめます。
北海道福島ほっかいどうふくしま町ちょうでは、人ひとが住すんでいる所ところに熊くまが来こないように、電気でんきが流ながれる柵さくを置おきました。
柵さくは、人ひとが住すむ場所ばしょと山やまの間あいだに置おきました。
福島町ふくしまちょうでは、去年きょねん、新聞しんぶんを配くばっていた男性だんせいが、熊くまに襲おそわれて亡なくなっています。
福島町ふくしまちょうは「熊くまの事故じこが起おこらないようにしていきたいです」と話はなしています。

3️⃣ 東京都とうきょうと　公園こうえんで倒たおれそうな木きを調しらべている

東京とうきょうの世田谷区せたがやくにある砧公園きぬたこうえんでは、先月せんげつと今月こんげつ、桜さくらやコナラなどの木きが倒たおれました。1か月げつの間あいだに4回かい、倒たおれました。けがをした人ひともいます。
このため東京都とうきょうとは、ほかに倒たおれそうな木きがないか、9日ここのかから調しらべています。木きの専門家せんもんかが、高たかさが3m以上いじょうある5000本ぼんを調しらべます。倒たおれそうな木きがあったら、すぐに切きろうと考かんがえています。
先月せんげつまでの1年ねん間かんに、東京都とうきょうとの公園こうえんなどでは、木きが、86本ぽん、倒たおれました。
東京都とうきょうとは、これから、砧公園きぬたこうえん以外いがいの公園こうえんの木きも調しらべる予定よていです。

4️⃣ 岩手大学いわてだいがく「猫ねこがごはんを残のこすのは匂においに慣なれるからだ」

「なぜ、うちの猫ねこはごはんを残のこすのだろうか」と不思議ふしぎに思おもっている人ひとも多おおいと思おもいます。
岩手大学いわてだいがくの研究けんきゅうグループは、12匹ひきの猫ねこで、調しらべました。
6回かい、食事しょくじをあげます。
同おなじ食事しょくじをあげ続つづけると、猫ねこが食たべる量りょうは、だんだん減へります。
しかし、6回かい目め、別べつの食事しょくじに変かえると、食たべる量りょうが増ふえました。
同おなじ食事しょくじでも、違ちがう匂においに変かえると、食たべる量りょうは増ふえました。
研究けんきゅうグループは、猫ねこはお腹なかがいっぱいだからではなく、匂においに慣なれたから食たべなくなったと言いっています。
そして、「ペットフードをつくるときに、匂においについても考かんがえると、猫ねこの食事しょくじが豊ゆたかになると思おもいます」と話はなしています。
