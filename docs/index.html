気象庁きしょうちょうは、40℃以上いじょうになる日ひが増ふえたため、名前なまえをつけることを考かんがえました。
ウェブサイトでアンケートをした結果けっか、「酷暑こくしょ日び」がいちばん多おおくなりました。ひどく暑あつい日ひという意味いみです。
酷暑こくしょ日びと聞きいたら、熱中症ねっちゅうしょうなどにならないように対策たいさくしてください。
