京都府南丹市きょうとふなんたんしの山やまで、亡なくなっていた小学生しょうがくせいの事件じけんについてです。
南丹市なんたんしの安達あだち結希ゆきさんは、今月こんげつ13日にちに、山やまの中なかで亡なくなっているのが見みつかりました。
警察けいさつは16日にち、結希ゆきさんの父親ちちおやの安達あだち優季ゆうき容疑者ようぎしゃを逮捕たいほしました。
警察けいさつによると、安達あだち容疑者ようぎしゃは、亡なくなった結希ゆきさんを山やまに運はこんで隠かくした疑うたがいがあります。そして、「私わたしがやったことに間違まちがいありません」と話はなして、認みとめています。
結希ゆきさんを殺ころしたことについても認みとめているようです。

2️⃣ 熊本地震くまもとじしんから10年ねん　県けんと全部ぜんぶの市しなどが初はじめて一緒いっしょに祈いのった

10年ねん前まえの2016年ねん4月がつ、熊本県くまもとけんでとても大おおきい地震じしんが2回かいありました。
2回かい目めの地震じしんから、16日にちで10年ねんになります。この地震じしんでは、熊本県くまもとけんと大分県おおいたけんで278人にんが亡なくなりました。
16日にち、熊本県くまもとけんと県けんにある全部ぜんぶの市しや町まちなどが、地震じしんのあと初はじめて一緒いっしょに、亡なくなった人ひとたちのために祈いのる式しきを行おこないました。260人にんくらいが集あつまって、1分ぷん間かん、静しずかに祈いのりました。そして、地震じしんのとき小学生しょうがくせいだった4人ひとが「避難所ひなんじょでやさしくしてもらって、安心あんしんすることができました。今度こんどは私わたしたちが助たすけることができるようになりたいと思おもいます」と話はなしました。

3️⃣ 病院びょういんで使つかう手袋てぶくろやごみの袋ふくろが少すくなくなっている

アメリカなどとイランの戦たたかいで、原油げんゆの輸入ゆにゅうが難むずかしくなっています。この影響えいきょうで、仕事しごとや生活せいかつで使つかう物ものが少すくなくなっています。
アスクルという会社かいしゃは、インターネットでいろいろな物ものを売うっています。病院びょういんで使つかう手袋てぶくろや、ごみの袋ふくろなどが、少すくなくなっています。このため、1人ひとりが買かうことができる数かずを決きめている物ものもあります。アスクルは、ウェブサイトを見みてほしいと言いっています。
そして、ライオンという会社かいしゃは、今年ことしの夏なつから、新あたらしい洗剤せんざいを売うる予定よていでした。洗濯せんたくの洗剤せんざいには石油せきゆからできる「ナフサ」を使つかっています。しかし、イランの問題もんだいがどうなるかわからないため、新あたらしい洗剤せんざいは予定よていよりあとに売うることを決きめました。

4️⃣ 立山黒部たてやまくろべアルペンルート　春はるが来きて通とおることができるようになった

春はるを感かんじるニュースです。
「立山黒部たてやまくろべアルペンルート」は、長野県ながのけんと富山県とやまけんの間あいだの37kmぐらいの道みちです。
山やまの中なかにあって、冬ふゆの間あいだは、たくさんの雪ゆきで通とおることができませんでした。
春はるが来きて、雪ゆきの片かたづけが終おわって、15日にちから通とおることができるようになりました。
愛知県あいちけんから来きた男性だんせいは「最初さいしょの日ひに来くることができて良よかったです。すばらしい景色けしきを楽たのしみたいです」と話はなしていました。
