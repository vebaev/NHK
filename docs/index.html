18日にちは、車くるまや機械きかいなどの大おおきい会社かいしゃが、働はたらく人ひとたちに答こたえる日ひでした。そして、平均へいきんで5.1％高たかくすると答こたえました。働はたらく人ひとたちの希望きぼうどおりに給料きゅうりょうを上あげると答こたえた会社かいしゃが、たくさんありました。
小ちいさい会社かいしゃは、給料きゅうりょうをどれくらいにするか、これから答こたえます。
