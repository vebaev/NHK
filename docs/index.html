アメリカとイスラエルがイランへ攻撃こうげきを始はじめてから10日とおかが過すぎました。

アメリカとイスラエルがイランへ攻撃こうげきを始はじめてから10日とおかが過すぎました。
トランプ大統領だいとうりょうは9日ここのか、今いままでにたくさんのことができたと言いいました。5000以上いじょうの狙ねらった目標もくひょうを攻撃こうげきして、イランの軍ぐんの船ふねやミサイルなどをたくさん壊こわしたと言いいました。
イランの近ちかくの海うみでは、原油げんゆを運はこぶ船ふねが通とおることが、難むずかしくなっています。
トランプ大統領だいとうりょうはSNSに「もし、原油げんゆを運はこぶ船ふねをイランが止とめたら、もっと大おおきい攻撃こうげきをする」と書かきました。
イランの軍ぐんの人ひとは「イランを攻撃こうげきする国くにとそれを助たすける国くには原油げんゆを1Lりっとるも運はこぶことができない」と言いっています。
戦たたかいが続つづいています。

東日本大震災ひがしにほんだいしんさいが起おきてから、11日にちで15年ねんです。

東日本大震災ひがしにほんだいしんさいが起おきてから、11日にちで15年ねんです。
宮城県気仙沼市みやぎけんけせんぬましでは、津波つなみで壊こわれた学校がっこうの建物たてものを今いまも大切たいせつに残のこしています。
10日とおか、気仙沼市けせんぬましの高校生こうこうせいなどが、この建物たてものを見みに来きた人ひとたちに津波つなみの話はなしをしました。
大震災だいしんさいのとき2歳さいだった高校生こうこうせいの佐藤さとう憂ゆ奈なさんは、家族かぞくから聞きいた15年ねん前まえのことや自分じぶんで勉強べんきょうしたことを話はなしました。そして、すぐに逃にげることの大切たいせつさを伝つたえました。
話はなしを聞きいた男性だんせいは「これからは、地震じしんや津波つなみは自分じぶんにも関係かんけいがあることだと考かんがえていきたいです」と話はなしました。

旅行りょこうや仕事しごとで外国がいこくに行いく時ときにはパスポートが必要ひつようです。

旅行りょこうや仕事しごとで外国がいこくに行いく時ときにはパスポートが必要ひつようです。
政府せいふは、法律ほうりつを変かえてパスポートを作つくる時ときに払はらうお金かねを今いままでより安やすくしたいと考かんがえています。
10年ねん使つかうことができるものを、今いまより7000円えん安やすい9000円えんぐらいにするつもりです。
18歳さい以上いじょうの場合ばあい、今いまは5年ねん使つかうことができるものもありますが、10年ねんのものだけになります。
政府せいふは、パスポートを作つくる時ときのお金かねを安やすくして外国がいこくに行いきやすくしたいと考かんがえています。
いまの国会こっかいで法律ほうりつが変かわると、7月がつ1日ついたちから安やすくなる予定よていです。

ミラノ・コルティナパラリンピックで、日本にっぽんの選手せんしゅが初はじめてメダルを取とりました。

ミラノ・コルティナパラリンピックで、日本にっぽんの選手せんしゅが初はじめてメダルを取とりました。
村岡桃佳むらおかももか選手せんしゅです。村岡むらおか選手せんしゅは、アルペンスキーの女子じょしスーパー大回転だいかいてんに出でました。座すわって滑すべるスキーのクラスです。
この競技きょうぎは、右みぎや左ひだりにカーブしながら速はやいスピードで滑すべります。
村岡むらおか選手せんしゅは去年きょねん大おおきなけがをして2月がつに練習れんしゅうに戻もどったばかりでしたが、難むずかしいコースを上手うまく滑すべりました。そして、銀ぎんメダルを取とりました。
村岡むらおか選手せんしゅは「無事ぶじに滑すべることができてよかったです。自分じぶんを褒ほめたい」と話はなしています。
