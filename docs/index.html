1️⃣ 京都府南丹市きょうとふなんたんし　人ひとが亡なくなっているのが見みつかった

警察けいさつによると、13日にち、京都府南丹市きょうとふなんたんしで、人ひとが亡なくなっているのが見みつかりました。
子こどものようです。
南丹市なんたんしでは、3月がつ23日にちから小学生しょうがくせいの安達あだち結希ゆきさんがいなくなって、警察けいさつが探さがしていました。
亡なくなった人ひとが見みつかったのは、小学校しょうがっこうから南みなみに2キロぐらいのところです。
警察けいさつによると、結希ゆきさんは、3月がつ23日にちの朝あさ、父親ちちおやが車くるまで小学校しょうがっこうに送おくったあと、いなくなりました。
29日にちには、リュックサックが見みつかりました。
そして12日にち、結希ゆきさんの家いえと小学校しょうがっこうの間あいだにある山やまの中なかで、子こどもの靴くつが見みつかりました。
警察けいさつは、亡なくなった人ひとが誰だれなのか、調しらべています。

2️⃣ アメリカとイランの話はなし合あいはうまくいかなかった

アメリカなどとイランの戦たたかいについてのニュースです。
アメリカとイランは、2週間しゅうかん、戦たたかいを止とめています。
11日にちと12日にち、2ふたつの国くにの代表だいひょうがパキスタンで会あいました。パキスタンは、戦たたかいを終おわりにしたいと考かんがえていて、一緒いっしょに話はなしをしました。しかし、話はなし合あいは、うまくいきませんでした。
話はなし合あいのあと、アメリカのバンス副ふく大統領だいとうりょうは「アメリカの考かんがえをイランが認みとめるかどうか見みていきたい」と言いいました。
イランの外務省がいむしょうは「意見いけんが大おおきく違ちがう問題もんだいがいくつかあった。一度いちどの話はなし合あいで同おなじ意見いけんになると考かんがえないほうがいい」と記者きしゃたちに言いいました。

3️⃣ トランプ大統領だいとうりょう　イランの港みなとを使つかう船ふねをアメリカがチェックする

アメリカとイランの話はなし合あいのあと、トランプ大統領だいとうりょうは「イランが石油せきゆを売うることができないようにする」と言いいました。
アメリカは、ホルムズ海峡かいきょうを通とおってイランの港みなとを使つかう船ふねをチェックします。
海峡かいきょうを通とおる船ふねから、イランがお金かねを取とることなども認みとめません。
日本にっぽんの時間じかんの13日にち、午後ごご11時じから始はじめると発表はっぴょうしました。
アメリカの軍ぐんは、イラン以外いがいの国くにの船ふねは、ホルムズ海峡かいきょうを自由じゆうに通とおることができると言いっています。
イランは、軍ぐんの船ふねが海峡かいきょうに近ちかづいたら、戦たたかいを止とめる約束やくそくと違ちがうと言いっています。
アメリカとイランの話はなし合あいがどうなるか、世界せかいの人ひとたちが心配しんぱいしています。

4️⃣ 秋田県あきたけん　赤あかちゃんホッキョクグマを見みることができるようにした

秋田県男鹿市あきたけんおがしの水族館すいぞくかんで、去年きょねん12月がつにホッキョクグマの赤あかちゃんが生うまれました。
赤あかちゃんはオスです。体重たいじゅうが19kgぐらいになって、外そとにいることに慣なれました。
そして11日にちから、水族館すいぞくかんは赤あかちゃんを、だれでも見みることができるようにしました。
赤あかちゃんはおもちゃで遊あそんだり、お母かあさんを追おいかけたりしています。
赤あかちゃんの名前なまえを決きめる投票とうひょうもあります。水族館すいぞくかんに来きた人ひとは、6つある名前なまえから1ひとつ選えらんでいました。
5歳さいの女おんなの子こは「ふわふわで白しろくてとてもかわいかった」と話はなしていました。
