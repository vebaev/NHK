アメリカのトランプ大統領だいとうりょうは、1日ついたちの夜よる、アメリカの人ひとたちに向むけて話はなしをしました。

2️⃣ 月つきを調しらべる宇宙船うちゅうせん　アメリカで打うち上あげ成功せいこう

アメリカは、人ひとが月つきに行いって調しらべる「アルテミス計画けいかく」を日本にっぽんやヨーロッパなどと一緒いっしょに進すすめています。
日本にっぽんの時間じかんの2日ふつか午前ごぜん7時半じはんすぎ、アメリカのフロリダから、宇宙船うちゅうせんをロケットで打うち上あげました。
宇宙船うちゅうせんに人ひとが乗のるのは、この計画けいかくでは初はじめてです。今回こんかいは4人にんが乗のって、10日とおか間かんで月つきのまわりを回まわって、戻もどる予定よていです。月つきに近ちかづいて写真しゃしんを撮とったり、地球ちきゅうとの通信つうしんに問題もんだいがないかなどを調しらべたりします。
最後さいごに人ひとが月つきに行いったのは、1972年ねんのアポロ17号ごうでした。

3️⃣ 秋田あきた市役所しやくしょ　熊くまの被害ひがいをなくすため「ハンター」を雇やとった

秋田県秋田市あきたけんあきたしでは去年きょねん、人ひとが熊くまに襲おそわれてけがをする事故じこがたくさんありました。
このため、市役所しやくしょは、銃じゅうを使つかった「ハンター」の仕事しごとができる人ひとを3人にん雇やといました。
30歳さいから59歳さいまでの秋田県あきたけんと福島県ふくしまけん、東京都とうきょうとの男性だんせいです。今いままでに熊くまを捕つかまえたことがあります。
3人にんは、これから、熊くまを捕つかまえるわなを置おいたり、銃じゅうを使つかって熊くまを撃うったりします。
男性だんせいの1人ひとりは「自分じぶんの技術ぎじゅつが役やくに立たつと思おもいました。みんなが安心あんしんして生活せいかつできればいいと思おもいます」と話はなしていました。
