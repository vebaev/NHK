1️⃣ アメリカとイスラエルがイランへの攻撃こうげきを始はじめてから1か月げつたった

アメリカとイスラエルがイランへの攻撃こうげきを始はじめてから28日にちで1か月げつになりました。
首都しゅとテヘランなどを空そらから攻撃こうげきしています。アメリカにある人権じんけんの団体だんたいなどは、イランで23日にちまでに1443人にん以上いじょうの市民しみんが亡なくなったと言いっています。
イランも、イスラエルなどへの攻撃こうげきを続つづけています。
戦たたかいを止とめるために話はなし合あいが必要ひつようです。トランプ大統領だいとうりょうは、29日にち、イランとの話はなし合あいは進すすんでいる、と記者きしゃたちに言いいました。
イランの隣となりにある国くに、パキスタンは、アメリカとイランとの話はなし合あいをパキスタンで開ひらいてもいいと言いっています。話はなし合あいがどうなるか、世界せかいが注目ちゅうもくしています。

2️⃣ 京都府きょうとふ　小学しょうがく5年生ねんせいの男おとこの子こがいなくなった

小学しょうがく5年生ねんせいの男おとこの子こがいなくなりました。
男おとこの子こは、京都府南丹市きょうとふなんたんしに住すんでいる安達あだち結希ゆきさん11歳さいです。
警察けいさつによると、お父とうさんは、23日にちの朝あさ、車くるまで結希ゆきさんを学校がっこうに送おくりました。そのあと、どこへ行いったかわかりません。
29日にち、学校がっこうから3kmぐらい離はなれた山やまの中なかで、結希ゆきさんのリュックサックが見みつかりました。警察けいさつは、山やまの中なかなどに結希ゆきさんがいないか、さがしています。

3️⃣ 西日本にしにほんと東日本ひがしにほん　雨あめや風かぜに気きをつけて

30日にちから31日にちまで、西日本にしにほんと、東日本ひがしにほんの太平洋たいへいよう側がわでとてもたくさんの雨あめが降ふったり急きゅうに強つよい風かぜが吹ふいたりするかもしれません。
気象庁きしょうちょうによると、雨あめを降ふらせる雲くもが西日本にしにほんから東日本ひがしにほんに動うごいています。天気てんきが悪わるくなります。
とくに九州きゅうしゅうや四国しこくなどでは雷かみなりが鳴なって、雨あめがたくさん降ふるところがあるかもしれません。
建物たてものが低ひくい所ところにあると水みずが入はいるかもしれません。
気象庁きしょうちょうは、急きゅうに冷つめたい風かぜが吹ふいたり雷かみなりの音おとが聞きこえたりしたら、すぐに丈夫じょうぶな建物たてものの中なかに入はいってほしいと言いっています。

4️⃣ 奈良県ならけん　いちばんいい金魚きんぎょを決きめる会かいがあった

泳およいでいるのを見みて楽たのしむ魚さかな、金魚きんぎょのニュースです。
奈良県ならけんで、28日にちいちばんいい金魚きんぎょを決きめる会かいがありました。
大和郡山市やまとこおりやましの金魚きんぎょは昔むかしから有名ゆうめいです。1年ねん間かんに4000万まん匹びきぐらいを育そだてています。
金魚きんぎょを育そだてている人ひとたちは100匹びきぐらいの金魚きんぎょを持もって集あつまりました。
いちばんいい賞しょうに選えらばれたのは体からだの長ながさが30cmぐらいの大おおきい金魚きんぎょです。赤あかと白しろ、そして黒くろの模様もようがあってきれいです。
大阪市おおさかしから来きた女おんなの人ひとは「初はじめて来きました。大おおきくてきれいな金魚きんぎょで、すごかったです」と話はなしていました。
